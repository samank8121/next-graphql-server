## Getting Started
Add a `.env` file with the following keys:
1. `CONNECTION_STRING` - Connection to the database
2. `TOKEN_SECRET_KEY` - Secret key for JSON Web Token

To start the development server, run:
```bash
yarn watch & yarn dev
